# XRPL Challenge 2: On-Chain Evidence Registration

## Challenge Overview
Trustbond.app is a blockchain application that helps NGOs raise donations transparently and prove their real-world actions by recording immutable evidence on the XRPL. This increases donor trust, using features like trustlines for issued currencies, on-chain evidence for accountability, and vaults (escrows) for conditional fund release.

Based on Ledger Structure (Day 1). Develop a mechanism to register evidence (e.g., hashes of photos/reports) using memos in transactions or hashes on ledger.

## What to Build
- Code transactions with memos for evidence.
- Integrate with DEX/AMM for linking to donations.
- Business: Build a simple dashboard (console or basic HTML) for on-chain data visualization.

## Setup
1. Clone this repo: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Run: `node index.js` (update with your wallet details).
